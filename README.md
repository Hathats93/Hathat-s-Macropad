| DSA Keycaps | 11 | Compatible with MX switches | 
| Seeed XIAO RP2040 | 1 | Microcontroller | 
| EC11 Rotary Encoder | 1 | Volue control | 
| PCB | 1 | Double-sided | 
| Case | 1 | 3D printed |
